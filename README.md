# Student Registration System

This is a web-based Student Registration Form developed using HTML, CSS, PHP, and MySQL.

## Features
- Form validation using PHP
- Error messages for invalid inputs
- Data stored in MySQL database
- Password stored securely using hashing

## Technologies Used
- HTML
- CSS
- PHP
- MySQL (XAMPP)

## How to Run
1. Install XAMPP
2. Start Apache and MySQL
3. Place project in htdocs folder
4. Create database in phpMyAdmin
5. Run using localhost
